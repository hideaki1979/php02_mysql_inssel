# ⑤工夫した点・こだわった点
- CSVから一括登録を行えるようにしました。  
検索画面はプルダウン表示の項目をDBから取得して表示するようにし、書籍情報TBLをベースに出版社を別テーブルで管理をして、SQLではINNER JOINを使って名称を取るようにしました。
